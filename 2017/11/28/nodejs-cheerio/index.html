Cheerio, a server side jQuery like tool to help work with HTML in nodejs

I have been making some personal projects that have to do with working with html in a nodejs environment. Often I am in a situation in which I end up generating html from markdown using something liked marked, or looping over actual html files, in ether case I have raw html that I want to work with in javaScript.

I have written a post recently on another node.js solution for this sort of thing called html-to-json, however in this post I will be writing about cheerio. Which is a server side implementation of core jQuery.

Selectors

Selectors work in very much the same way as in jQuery. The ushual selection by tag, class, and id work in very much the same fashion.

each method

A method that I actualy use often is for course the each method that loops over all elements in a collection.
