lodash deburr and latin 1 Unicode car removal

When it comes to Unicode blocks there are the first few blocks that have to do with Latin characters. These kinds of characters come up now and then for cretin words that come from languages like Spanish. If for some reason I might be interested in just simply converting these kinds of strings into a string that contains just the first few ASCCI range characters I can used the lodash deburr method. This method in lodash just simply takes away any additional accent over a letter and just converts into a plain English style form of the word.

1 - Latin 1 Supplement, Latin Extended A, and Unicode blocks in general

The lodash deburr method has to do with converting certain characters in the Unicode Blocks called Latin 1 Supplement#Character_table) and Latin A Extended Unicode into basic Latin characters that are found within plain old ASCII or the Basic Latin Unicode code block).
