Javascript style api for changing CSS values.

The JavaScript style api is one way to go about changing css values with javaScript rather than static hard coded CSS. This is not to be confused with javaScript coding style, which is of course a whole other subject. There are other ways of changing CSS values with javaScript such as changing the className property value of one or more elements with respect to a collection of hard coded CSS classes to work with. The style api is not the best choice for doing anything that might involve complex animations, or a great deal of rapid fast change for that there are canvas elements and SVG to work with.
